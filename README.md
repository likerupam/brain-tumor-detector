Brain Tumor Detector (CNN)

**Simple Convolutional Neural Network** to classify brain MRI images as Tumor vs No Tumor.  
This project demonstrates data loading, preprocessing, training, evaluation, and single-image prediction using a Kaggle brain MRI dataset.

Project Overview

- **Task:** Binary classification (Tumor / No Tumor) from brain MRI images.  
- **Model:** Convolutional Neural Network (TensorFlow / Keras).
